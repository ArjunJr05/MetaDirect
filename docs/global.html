(async) handleFetchLinkPreview(event, url) → {Promise.<Object>}

Handler for the 'fetch-link-preview' IPC event. Performs a secure, client-side metadata extraction with the following optimizations: - In-memory caching: Prevents redundant network requests for the same URL. - Partial Fetching: Aborts the request once the HTML or 128KB limit is reached. - Image Normalization: Resolves relative image paths to absolute URLs.
